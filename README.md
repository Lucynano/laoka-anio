### 🍽️ Laoka-Anio

Laoka-Anio est un petit programme développé en C++ qui génère quotidiennement un plat malagasy de manière aléatoire.

Il permet de gagner du temps en évitant la question quotidienne :

"Inona no laoka androany ?"


Pour le compiler: 

```bash
g++ laoka_anio.cpp -o test
```

Pour executer:

```bash
./test
```
